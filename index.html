Essential tools for recon, exploitation, and analysis.

Nmap (Network Mapper)

The go-to tool for network discovery and security auditing. It scans networks to identify active hosts, open ports, service versions, and operating systems.

Metasploit Framework

A premier exploitation framework used to find, exploit, and validate vulnerabilities. It includes a massive database of prebuilt exploits and payloads, making it essential for penetration testing.

Burp Suite

A comprehensive platform for web application security testing. It acts as an intercepting proxy, allowing users to analyze, modify, and replay web traffic to identify SQL injection, XSS, and other vulnerabilities.

Wireshark

The world's foremost network protocol analyzer. It captures network traffic in real-time, allowing for deep inspection of hundreds of protocols to troubleshoot or analyze suspicious activity.

Aircrack-ng

A complete suite of tools for assessing WiFi network security. It specializes in monitoring, attacking (de-authentication), testing, and cracking (WEP/WPA/WPA2-PSK) wireless networks.

Sqlmap

An automated tool for detecting and exploiting SQL injection flaws in web applications. It streamlines database takeover, data extraction, and user privilege escalation.

Hydra (THC-Hydra)

A very fast network logon cracker. It performs rapid dictionary attacks against more than 50 protocols, including SSH, Telnet, FTP, HTTP, and RDP.

Hashcat

The world's fastest password recovery tool, leveraging GPU power to crack complex password hashes, making it faster than CPU-based tools.

Bettercap

A powerful, flexible tool for network reconnaissance and man-in-the-middle (MitM) attacks. It is effective for sniffing, spoofing, and attacking both wired and wireless networks.

Social-Engineer Toolkit (SET)

Used to simulate social engineering attacks like phishing.

Netcat (nc)

The "Swiss army knife" of networking for reading/writing data across network connections.

John the Ripper

A fast, versatile, offline password-cracking tool. It is widely used to identify weak passwords by testing various encryption formats, hashes, and dictionary attacks.
